# E0001: Source File Read Failure

## Explanation

The compiler or project driver could not read a `.nomo` source file from disk.
This is an I/O failure before lexing or parsing starts.

## Error Example

```console
nomoc check missing.nomo
```

## Correct Code

```console
nomoc check src/main.nomo
```

## Fix

Check that the path exists, points to a readable file, and is accessible to the
current process. For project commands, also check that `src/main.nomo` exists
and that imported module files can be read.

# E0101: Unterminated String Literal

## Explanation

The lexer reached the end of a source line before finding the closing `"` for a
string literal.

## Error Example

```nomo
package app.main

fn main() -> void {
    let message: string = "hello
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    let message: string = "hello"
}
```

## Fix

Add the closing `"` or rewrite the string so it fits on one supported source
line. Use escapes such as `\n`, `\r`, `\t`, `\"`, and `\\` for special
characters.

## LSP Code Action

No automatic code action is currently provided.

# E0102: Unexpected Character

## Explanation

The lexer found a character that is not part of the Nomo v0.1 token set.

## Error Example

```nomo
package app.main

fn main() -> void {
    @
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    let value: i64 = 1
}
```

## Fix

Remove the unsupported character or replace it with a valid Nomo token.

## LSP Code Action

No automatic code action is currently provided.

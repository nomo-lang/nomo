# E0105: Unknown Character Escape

## Explanation

A character literal used an escape sequence that the lexer does not recognize.
Nomo v0.1 currently accepts `\n`, `\r`, `\t`, `\'`, and `\\` in character
literals.

## Error Example

```nomo
package app.main

fn main() -> void {
    let value: char = '\x'
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    let value: char = '\n'
}
```

## Fix

Use one of the supported escapes, or write the character directly when it is a
single valid source character.

## LSP Code Action

No automatic code action is currently provided.

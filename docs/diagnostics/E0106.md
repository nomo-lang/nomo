# E0106: Empty Character Literal

## Explanation

A character literal opened and closed immediately without containing a
character.

## Error Example

```nomo
package app.main

fn main() -> void {
    let value: char = ''
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    let value: char = 'x'
}
```

## Fix

Place exactly one character or one supported escape sequence between the single
quotes.

## LSP Code Action

No automatic code action is currently provided.

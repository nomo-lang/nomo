# E0107: Character Literal Has Too Many Characters

## Explanation

A character literal contained more than one source character before the closing
quote. Nomo character literals must contain exactly one character or one
supported escape sequence.

## Error Example

```nomo
package app.main

fn main() -> void {
    let value: char = 'ab'
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    let first: char = 'a'
    let second: char = 'b'
}
```

## Fix

Use a string literal for text, or keep the character literal to exactly one
character.

## LSP Code Action

No automatic code action is currently provided.

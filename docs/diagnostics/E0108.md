# E0108: Unterminated Block Comment

## Explanation

The lexer reached the end of the file while still inside a block comment.
Block comments use `/* ... */` and may be nested.

## Error Example

```nomo
package app.main

/* open comment
fn main() -> void {
}
```

## Correct Code

```nomo
package app.main

/* closed comment */
fn main() -> void {
}
```

## Fix

Add the missing closing `*/`. If comments are nested, make sure every nested
`/*` has its own matching `*/`.

No automatic code action is currently provided.

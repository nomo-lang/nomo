# E0202: Expected Function Declaration

## Explanation

The parser entered a function declaration parse path but did not find the `fn`
keyword at the declaration start.

## Error Example

```nomo
package app.main

pub main() -> void {
}
```

## Correct Code

```nomo
package app.main

pub fn main() -> void {
}
```

## Fix

Add the `fn` keyword before the function name.

## LSP Code Action

No automatic code action is currently provided.

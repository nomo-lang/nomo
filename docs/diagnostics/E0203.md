# E0203: Missing Function Parameter List

## Explanation

After a function name and any type parameters, the parser expects `(` to start
the parameter list.

## Error Example

```nomo
package app.main

fn add a: i64, b: i64 -> i64 {
    return a + b
}
```

## Correct Code

```nomo
package app.main

fn add(a: i64, b: i64) -> i64 {
    return a + b
}
```

## Fix

Wrap function parameters in parentheses. Use `()` when a function takes no
parameters.

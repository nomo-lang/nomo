# E0206: Missing Function Body

## Explanation

The parser reached the end of a function signature but did not find `{` to
start the function body.

## Error Example

```nomo
package app.main

fn main() -> void
    return
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    return
}
```

## Fix

Add `{` after the return type or parameter list, then put the function
statements inside the body.

No automatic code action is currently provided.

# E0207: Unterminated Function Body

## Explanation

The parser reached the end of the file while parsing a function body. Function
bodies must close with `}`.

## Error Example

```nomo
package app.main

fn main() -> void {
    return
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    return
}
```

## Fix

Add the missing `}`. If the body contains nested blocks, make sure every nested
`{` has a matching `}`.

No automatic code action is currently provided.

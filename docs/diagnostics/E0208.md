# E0208: Expected Expression

## Explanation

The parser expected an expression but found a token that cannot start one. This
often happens after `return`, after `=`, inside parentheses, or in function
call arguments.

## Error Example

```nomo
package app.main

fn answer() -> i64 {
    return +
}
```

## Correct Code

```nomo
package app.main

fn answer() -> i64 {
    return 1 + 2
}
```

## Fix

Replace the unexpected token with a valid expression such as a literal, name,
call, parenthesized expression, `if`, `match`, or `panic`.

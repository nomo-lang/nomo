# E0209: Missing Closing Parenthesis

## Explanation

The parser read a parenthesized expression but did not find the closing `)`.

## Error Example

```nomo
package app.main

fn answer() -> i64 {
    return (1 + 2
}
```

## Correct Code

```nomo
package app.main

fn answer() -> i64 {
    return (1 + 2)
}
```

## Fix

Add the missing `)` after the parenthesized expression.

No automatic code action is currently provided.

# E0210: Invalid Call Argument List

## Explanation

The parser found an invalid function call shape. This can happen when generic
call type arguments are not followed by `(`, or when a call argument list is not
closed with `)`.

## Error Example

```nomo
package app.main

fn main() -> void {
    print<string>
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    print<string>("hello")
}
```

## Fix

Add `(` after generic call type arguments and close every call argument list
with `)`.

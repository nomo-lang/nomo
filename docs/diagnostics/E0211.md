# E0211: Missing Required Newline

## Explanation

Nomo v0.1 uses newlines to separate package declarations, imports,
declarations, and statements. The parser expected a newline but found another
token on the same line.

## Error Example

```nomo
package app.main fn main() -> void {
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
}
```

## Fix

Move the next declaration or statement onto its own line. Do not use semicolons
to separate statements in v0.1.

No automatic code action is currently provided.

# E0211: Missing Required Newline

## Explanation

Nomo v0.1 uses newlines to separate package declarations, imports,
declarations, and statements. The parser expected a newline but found another
token on the same line.

## Error Example

```nomo
package app.main fn main() -> void {
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
}
```

## Fix

Move the next declaration or statement onto its own line. Semicolons are only
valid between the initializer, condition, and update clauses of a three-clause
loop such as `for let i = 0; i < 10; i++ { ... }`; they do not separate
ordinary statements.

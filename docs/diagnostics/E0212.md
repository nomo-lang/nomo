# E0212: Invalid Let Statement

## Explanation

The parser was reading a `let` statement but did not find a valid binding name.
For ordinary `let`, the binding must be a single variable name.

## Error Example

```nomo
package app.main

fn main() -> void {
    let app.value = 1
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    let value = 1
}
```

## Fix

Use a single variable name after `let`. Use assignment syntax for existing
variables or fields.

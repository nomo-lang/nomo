# E0213: Missing Initializer

## Explanation

The parser expected `=` before a `let` initializer.

## Error Example

```nomo
package app.main

fn main() -> void {
    let value: i64 1
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    let value: i64 = 1
}
```

## Fix

Add `=` between the binding or type annotation and the initializer expression.

# E0214: Missing Parameter Type Separator

## Explanation

Function and method parameters use `name: Type`. The parser found a parameter
name but did not find `:` before its type.

## Error Example

```nomo
package app.main

fn add(a i64, b: i64) -> i64 {
    return a + b
}
```

## Correct Code

```nomo
package app.main

fn add(a: i64, b: i64) -> i64 {
    return a + b
}
```

## Fix

Add `:` after the parameter name, followed by the parameter type.

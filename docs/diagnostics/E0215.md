# E0215: Invalid Parameter Separator

## Explanation

After each function or method parameter, the parser expects either `,` for
another parameter or `)` to end the parameter list.

## Error Example

```nomo
package app.main

fn add(a: i64 b: i64) -> i64 {
    return a + b
}
```

## Correct Code

```nomo
package app.main

fn add(a: i64, b: i64) -> i64 {
    return a + b
}
```

## Fix

Add a comma between parameters, or close the list with `)` after the final
parameter.

# E0216: Expected Return Statement

## Explanation

The parser entered the return-statement parse path but did not find the
`return` keyword.

## Error Example

```nomo
package app.main

fn answer() -> i64 {
    ret 1
}
```

## Correct Code

```nomo
package app.main

fn answer() -> i64 {
    return 1
}
```

## Fix

Use the `return` keyword for explicit returns.

No automatic code action is currently provided.

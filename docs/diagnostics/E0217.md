# E0217: Invalid Assignment or Postfix Update

## Explanation

Assignments and postfix updates can target a variable or a direct field, but
not a longer path. The parser also reports this code if an assignment or
postfix update parse path does not contain the expected operator.

## Error Example

```nomo
package app.main

fn main() -> void {
    app.user.name = "Nomo"
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    user.name = "Nomo"
}
```

## Fix

Assign to a local variable or a direct field, and use a supported assignment
operator such as `=`, `+=`, or `-=`. For postfix updates, use `++` or `--` on a
variable or direct field.

# E0218: Expected Struct Declaration

## Explanation

The parser entered the struct declaration parse path but did not find the
`struct` keyword.

## Error Example

```nomo
package app.main

pub User {
    name: string
}
```

## Correct Code

```nomo
package app.main

pub struct User {
    name: string
}
```

## Fix

Add the `struct` keyword before the type name.

No automatic code action is currently provided.

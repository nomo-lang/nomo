# E0219: Missing Struct Body

## Explanation

After a struct name and any generic type parameters, the parser expects `{` to
start the field list.

## Error Example

```nomo
package app.main

struct User
    name: string
}
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}
```

## Fix

Add `{` before the struct fields.

No automatic code action is currently provided.

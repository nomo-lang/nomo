# E0220: Unterminated Struct Body

## Explanation

The parser reached the end of the file while parsing a struct body. Struct
bodies must close with `}`.

## Error Example

```nomo
package app.main

struct User {
    name: string
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}
```

## Fix

Add the missing `}` after the final field.

# E0221: Missing Struct Field Type Separator

## Explanation

Struct fields use `name: Type`. The parser found a field name but did not find
the required `:` before the field type.

## Error Example

```nomo
package app.main

struct User {
    name string
}
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}
```

## Fix

Add `:` after the field name, followed by the field type.

# E0222: Missing Struct Literal Body

## Explanation

The parser recognized a struct literal parse path but did not find `{` before
the literal fields.

## Error Example

```nomo
package app.main

struct User {
    name: string
}

fn make() -> User {
    return User name: "Nomo" }
}
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}

fn make() -> User {
    return User { name: "Nomo" }
}
```

## Fix

Add `{` after the struct type name before listing literal fields.

No automatic code action is currently provided.

# E0223: Missing Struct Literal Field Separator

## Explanation

Struct literal fields use `name: value`. The parser found a field name but did
not find `:` before the field value.

## Error Example

```nomo
package app.main

struct User {
    name: string
}

fn make() -> User {
    return User { name "Nomo" }
}
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}

fn make() -> User {
    return User { name: "Nomo" }
}
```

## Fix

Add `:` between the field name and value.

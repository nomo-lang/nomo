# E0225: Unterminated Struct Literal

## Explanation

The parser reached the end of a struct literal field list without finding the
closing `}`.

## Error Example

```nomo
package app.main

struct User {
    name: string
}

fn make() -> User {
    return User { name: "Nomo"
}
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}

fn make() -> User {
    return User { name: "Nomo" }
}
```

## Fix

Add the missing `}` after the struct literal fields.

# E0226: Expected Enum Declaration

## Explanation

The parser entered the enum declaration parse path but did not find the `enum`
keyword.

## Error Example

```nomo
package app.main

pub Status {
    Ready
}
```

## Correct Code

```nomo
package app.main

pub enum Status {
    Ready
}
```

## Fix

Add the `enum` keyword before the enum name.

## LSP Code Action

No automatic code action is currently provided.

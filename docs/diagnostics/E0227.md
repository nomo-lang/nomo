# E0227: Missing Enum Body

## Explanation

After an enum name and any generic type parameters, the parser expects `{` to
start the variant list.

## Error Example

```nomo
package app.main

enum Status
    Ready
}
```

## Correct Code

```nomo
package app.main

enum Status {
    Ready
}
```

## Fix

Add `{` before the enum variants.

## LSP Code Action

No automatic code action is currently provided.

# E0228: Unterminated Enum Body

## Explanation

The parser reached the end of the file while parsing an enum body. Enum bodies
must close with `}`.

## Error Example

```nomo
package app.main

enum Status {
    Ready
```

## Correct Code

```nomo
package app.main

enum Status {
    Ready
}
```

## Fix

Add the missing `}` after the final variant.

No automatic code action is currently provided.

# E0229: Expected Match Expression

## Explanation

The parser entered the match parse path but did not find the `match` keyword.

## Error Example

```nomo
package app.main

fn label(value: i64) -> string {
    return value {
        1 => "one"
    }
}
```

## Correct Code

```nomo
package app.main

fn label(value: i64) -> string {
    return match value {
        1 => "one"
    }
}
```

## Fix

Add the `match` keyword before the scrutinee expression.

No automatic code action is currently provided.

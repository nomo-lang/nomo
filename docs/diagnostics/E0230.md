# E0230: Missing Match Body

## Explanation

After the match scrutinee expression, the parser expects `{` to start the match
arms.

## Error Example

```nomo
package app.main

fn label(value: i64) -> string {
    return match value
        1 => "one"
}
```

## Correct Code

```nomo
package app.main

fn label(value: i64) -> string {
    return match value {
        1 => "one"
    }
}
```

## Fix

Add `{` before the match arms.

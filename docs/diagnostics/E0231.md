# E0231: Unterminated Match Body

## Explanation

The parser reached the end of the file while parsing a match body. Match bodies
must close with `}`.

## Error Example

```nomo
package app.main

fn label(value: i64) -> string {
    return match value {
        1 => "one"
```

## Correct Code

```nomo
package app.main

fn label(value: i64) -> string {
    return match value {
        1 => "one"
    }
}
```

## Fix

Add the missing `}` after the final match arm.

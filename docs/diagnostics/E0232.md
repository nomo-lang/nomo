# E0232: Missing Match Arm Arrow

## Explanation

Each match arm must use `=>` between the pattern and the arm body.

## Error Example

```nomo
package app.main

fn label(value: i64) -> string {
    return match value {
        1 "one"
    }
}
```

## Correct Code

```nomo
package app.main

fn label(value: i64) -> string {
    return match value {
        1 => "one"
    }
}
```

## Fix

Add `=>` after the match pattern.

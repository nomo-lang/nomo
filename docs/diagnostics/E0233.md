# E0233: Missing Enum Payload Closing Parenthesis

## Explanation

An enum variant payload type opened with `(` but did not close with `)`.

## Error Example

```nomo
package app.main

enum MaybeInt {
    Some(i64
    None
}
```

## Correct Code

```nomo
package app.main

enum MaybeInt {
    Some(i64)
    None
}
```

## Fix

Add the missing `)` after the enum variant payload type.

No automatic code action is currently provided.

# E0234: Invalid Match Binding

## Explanation

The parser expected a binding name in a match pattern or let-else pattern, or it
expected `)` after an existing match pattern binding.

## Error Example

```nomo
package app.main

fn label(value: Option<string>) -> string {
    return match value {
        Some(text => text
        None => "missing"
    }
}
```

## Correct Code

```nomo
package app.main

fn label(value: Option<string>) -> string {
    return match value {
        Some(text) => text
        None => "missing"
    }
}
```

## Fix

Use a valid binding name and close the binding with `)`.

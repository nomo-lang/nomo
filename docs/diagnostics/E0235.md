# E0235: Invalid Generic Parameter or Match Arm Body

## Explanation

The parser reports this code in two parser contexts:

- after a generic type parameter, where it expects `,` or `>`;
- in statement-form `match`, where it expects `{` before a match arm body.

## Error Example

```nomo
package app.main

struct Box<T U> {
    value: T
}
```

## Correct Code

```nomo
package app.main

struct Box<T, U> {
    value: T
}
```

For statement-form match arms:

```nomo
package app.main

fn print(value: Option<string>) -> void {
    match value {
        Some(text) => {
            println(text)
        }
    }
}
```

## Fix

Add `,` or `>` in generic parameter lists. For statement-form match arms, wrap
the arm body in `{ ... }`.

No automatic code action is currently provided.

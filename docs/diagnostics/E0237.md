# E0237: Duplicate Generic Type Parameter

## Explanation

A generic declaration reused the same type parameter name more than once in the
same parameter list.

## Error Example

```nomo
package app.main

struct Pair<T, T> {
    left: T
    right: T
}
```

## Correct Code

```nomo
package app.main

struct Pair<T, U> {
    left: T
    right: U
}
```

## Fix

Rename duplicate type parameters so each generic parameter in the list has a
unique name.

# E0238: Unsupported Match Wildcard

## Explanation

The parser found `_` in a match pattern or match binding. Wildcard match
patterns are intentionally not supported in v0.1.

## Error Example

```nomo
package app.main

fn label(value: Option<string>) -> string {
    return match value {
        _ => "anything"
    }
}
```

## Correct Code

```nomo
package app.main

fn label(value: Option<string>) -> string {
    return match value {
        Some(text) => text
        None => "missing"
    }
}
```

## Fix

Write explicit match arms for every supported variant instead of using `_`.

## LSP Code Action

No automatic code action is currently provided.

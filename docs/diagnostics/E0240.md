# E0240: Expected If Expression

## Explanation

The parser entered the `if` expression parse path but did not find the `if`
keyword.

## Error Example

```nomo
package app.main

fn label(ok: bool) -> string {
    return ok { "yes" } else { "no" }
}
```

## Correct Code

```nomo
package app.main

fn label(ok: bool) -> string {
    return if ok { "yes" } else { "no" }
}
```

## Fix

Add the `if` keyword before the condition expression.

No automatic code action is currently provided.

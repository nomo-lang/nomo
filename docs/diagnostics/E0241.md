# E0241: Missing If Branch Body

## Explanation

After an `if` expression condition, the parser expects `{` to start the then
branch expression block.

## Error Example

```nomo
package app.main

fn label(ok: bool) -> string {
    return if ok "yes" else { "no" }
}
```

## Correct Code

```nomo
package app.main

fn label(ok: bool) -> string {
    return if ok { "yes" } else { "no" }
}
```

## Fix

Wrap the then branch expression in `{ ... }`.

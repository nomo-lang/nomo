# E0242: Unterminated Expression Block

## Explanation

The parser read an expression block but did not find the closing `}`. Expression
blocks are used by `if` branches and some internal expression parse paths.

## Error Example

```nomo
package app.main

fn label(ok: bool) -> string {
    return if ok { "yes" else { "no" }
}
```

## Correct Code

```nomo
package app.main

fn label(ok: bool) -> string {
    return if ok { "yes" } else { "no" }
}
```

## Fix

Add the missing `}` after the expression block.

# E0244: Missing Else Branch

## Explanation

Nomo `if` expressions must be exhaustive: after the then branch, the parser
expects an `else` branch.

## Error Example

```nomo
package app.main

fn label(ok: bool) -> string {
    return if ok { "yes" }
}
```

## Correct Code

```nomo
package app.main

fn label(ok: bool) -> string {
    return if ok { "yes" } else { "no" }
}
```

## Fix

Add an `else` branch that produces the same expression type as the then branch.

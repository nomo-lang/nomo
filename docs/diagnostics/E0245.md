# E0245: Missing Else Branch Body

## Explanation

After the `else` keyword in an `if` expression, the parser expects `{` to start
the else branch expression block.

## Error Example

```nomo
package app.main

fn label(ok: bool) -> string {
    return if ok { "yes" } else "no"
}
```

## Correct Code

```nomo
package app.main

fn label(ok: bool) -> string {
    return if ok { "yes" } else { "no" }
}
```

## Fix

Wrap the else branch expression in `{ ... }`.

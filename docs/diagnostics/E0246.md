# E0246: Expected Panic Expression

## Explanation

The parser entered the panic expression parse path but did not find the `panic`
keyword.

## Error Example

```nomo
package app.main

fn main() -> void {
    fail("boom")
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    panic("boom")
}
```

## Fix

Use `panic(...)` when intentionally aborting execution, or call a regular
function if no panic is intended.

No automatic code action is currently provided.

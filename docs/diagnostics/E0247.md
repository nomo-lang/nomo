# E0247: Missing Panic Argument List

## Explanation

After the `panic` keyword, the parser expects `(` to start the panic message
expression.

## Error Example

```nomo
package app.main

fn main() -> void {
    panic "boom"
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    panic("boom")
}
```

## Fix

Wrap the panic message expression in parentheses.

No automatic code action is currently provided.

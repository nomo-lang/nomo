# E0248: Missing Panic Closing Parenthesis

## Explanation

The parser read a `panic(` message expression but did not find the closing `)`.

## Error Example

```nomo
package app.main

fn main() -> void {
    panic("boom"
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    panic("boom")
}
```

## Fix

Add the missing `)` after the panic message.

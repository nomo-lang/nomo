# E0250: Expected Impl Block

## Explanation

The parser entered the impl-block parse path but did not find the `impl`
keyword.

## Error Example

```nomo
package app.main

struct User {
    name: string
}

User {
    fn name(self) -> string {
        return self.name
    }
}
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}

impl User {
    fn name(self) -> string {
        return self.name
    }
}
```

## Fix

Add `impl` before the target type.

No automatic code action is currently provided.

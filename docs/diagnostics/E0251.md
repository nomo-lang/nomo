# E0251: Invalid Impl Target Type

## Explanation

The parser accepted an impl target shape that is outside the v0.1 boundary.
Impl blocks must target a local, non-generic type name.

## Error Example

```nomo
package app.main

struct Box<T> {
    value: T
}

impl Box<i64> {
    fn value(self) -> i64 {
        return self.value
    }
}
```

## Correct Code

```nomo
package app.main

struct Box {
    value: i64
}

impl Box {
    fn value(self) -> i64 {
        return self.value
    }
}
```

## Fix

Use a single local type name as the impl target. Generic impl targets are not
supported in v0.1.

## LSP Code Action

No automatic code action is currently provided.

# E0252: Missing Impl Body

## Explanation

After an impl target, the parser expects `{` to start the method list.

## Error Example

```nomo
package app.main

struct User {
    name: string
}

impl User
    fn name(self) -> string {
        return self.name
    }
}
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}

impl User {
    fn name(self) -> string {
        return self.name
    }
}
```

## Fix

Add `{` before impl methods.

No automatic code action is currently provided.

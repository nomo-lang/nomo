# E0253: Unterminated Impl Body

## Explanation

The parser reached the end of the file while parsing an impl body. Impl bodies
must close with `}`.

## Error Example

```nomo
package app.main

struct User {
    name: string
}

impl User {
    fn name(self) -> string {
        return self.name
    }
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}

impl User {
    fn name(self) -> string {
        return self.name
    }
}
```

## Fix

Add the missing `}` after the final method.

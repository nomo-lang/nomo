# E0254: Invalid Impl Body Member

## Explanation

Impl bodies may contain method declarations. The parser found a token that is
neither a method declaration nor the closing `}`.

## Error Example

```nomo
package app.main

struct User {
    name: string
}

impl User {
    let value = 1
}
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}

impl User {
    fn name(self) -> string {
        return self.name
    }
}
```

## Fix

Use `fn` method declarations inside impl blocks, or move non-method statements
into a method body.

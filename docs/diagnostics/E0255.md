# E0255: Unsupported Impl Target

## Explanation

The compiler rejected an impl target that is outside the current v0.1
implementation boundary. Impl blocks must target local, non-generic structs.

## Error Example

```nomo
package app.main

enum Status {
    Ready
}

impl Status {
    fn label(self) -> string {
        return "ready"
    }
}
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}

impl User {
    fn label(self) -> string {
        return self.name
    }
}
```

## Fix

Move methods onto a local non-generic struct. Enum impls, external type impls,
and generic impl targets are not supported in v0.1.

## LSP Code Action

No automatic code action is currently provided.

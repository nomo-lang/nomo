# E0256: Invalid Method Self Parameter

## Explanation

Methods inside impl blocks must declare `self` as their first parameter. The
compiler reports this error when `self` is missing or appears in the wrong
position.

## Error Example

```nomo
package app.main

struct User {
    name: string
}

impl User {
    fn label() -> string {
        return "user"
    }
}
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}

impl User {
    fn label(self) -> string {
        return self.name
    }
}
```

## Fix

Add `self` as the first method parameter.

## LSP Code Action

No automatic code action is currently provided.

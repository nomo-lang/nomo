# E0257: Invalid Method Receiver Type

## Explanation

The compiler found a method receiver whose type does not match the impl target,
or a method call receiver that does not match the method signature.

## Error Example

```nomo
package app.main

struct User {
    name: string
}

struct Team {
    name: string
}

impl User {
    fn label(self: Team) -> string {
        return self.name
    }
}
```

## Correct Code

```nomo
package app.main

struct User {
    name: string
}

impl User {
    fn label(self) -> string {
        return self.name
    }
}
```

## Fix

Use `self` without an explicit type when possible, or make the explicit receiver
type match the impl target exactly.

## LSP Code Action

No automatic code action is currently provided.

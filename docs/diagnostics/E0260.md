# E0260: Expected For Statement

## Explanation

The parser entered the `for` statement parse path but did not find the `for`
keyword where the statement must start.

## Error Example

```nomo
package app.main

fn main() -> void {
    true {
        return
    }
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    for true {
        return
    }
}
```

## Fix

Start loop statements with `for`.

No automatic code action is currently provided.

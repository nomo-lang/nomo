# E0261: Missing Infinite For Body

## Explanation

An infinite loop is written as `for { ... }`. The parser expected `{` to start
the loop body.

## Error Example

```nomo
package app.main

fn main() -> void {
    for
        return
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    for {
        return
    }
}
```

## Fix

Add `{ ... }` immediately after `for` for an infinite loop body.

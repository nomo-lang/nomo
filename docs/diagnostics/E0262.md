# E0262: Missing For-In Separator

## Explanation

Iterator loops use `for name in iterable { ... }`. The parser expected the `in`
separator after the loop binding.

## Error Example

```nomo
package app.main

fn main() -> void {
    for item values {
        return
    }
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    for item in values {
        return
    }
}
```

## Fix

Insert `in` between the loop binding and the iterable expression.

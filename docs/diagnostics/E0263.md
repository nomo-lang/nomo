# E0263: Missing For-In Body

## Explanation

After `for name in iterable`, the parser expects `{` to start the loop body.

## Error Example

```nomo
package app.main

fn main() -> void {
    for item in values
        return
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    for item in values {
        return
    }
}
```

## Fix

Wrap the for-in loop body in `{ ... }`.

# E0264: Missing Conditional For Body

## Explanation

Conditional loops use `for condition { ... }`. After the condition expression,
the parser expected `{` to start the loop body.

## Error Example

```nomo
package app.main

fn main() -> void {
    for keep_running
        return
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    for keep_running {
        return
    }
}
```

## Fix

Add `{ ... }` after the loop condition.

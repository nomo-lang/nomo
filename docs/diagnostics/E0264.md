# E0264: Invalid For Header or Missing Body

## Explanation

Conditional loops use `for condition { ... }`. Three-clause loops use
`for let binding = initializer; condition; update { ... }`. The parser could
not find a required clause separator or the `{` that starts the loop body.

## Error Example

```nomo
package app.main

fn main() -> void {
    for keep_running
        return
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    for keep_running {
        return
    }
}
```

## Fix

Add `{ ... }` after the loop condition.

For a three-clause loop, also ensure both `;` separators are present and the
update assigns to, increments, or decrements the loop binding.

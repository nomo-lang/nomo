# E0265: Expected Defer Statement

## Explanation

The parser or lowering step expected a `defer` statement. A `defer` statement
starts with `defer` followed by the statement to run when the current scope
exits.

## Error Example

```nomo
package app.main

fn main() -> void {
    cleanup()
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    defer cleanup()
}
```

## Fix

Add `defer` before the cleanup statement, or move the statement out of a context
that requires deferred execution.

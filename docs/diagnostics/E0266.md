# E0266: Unterminated Statement Block

## Explanation

The parser reached the end of the file while reading a statement block. Blocks
opened with `{` must close with `}`.

## Error Example

```nomo
package app.main

fn main() -> void {
    for {
        return
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    for {
        return
    }
}
```

## Fix

Add the missing `}` for the innermost statement block.

No automatic code action is currently provided.

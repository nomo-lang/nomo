# E0267: Missing Let-Else Else or Const Declaration

## Explanation

This parser code is used in two declaration contexts:

- a `let Pattern(binding) = value` pattern binding must be followed by `else`;
- a top-level constant declaration must start with `const`.

## Error Example

```nomo
package app.main

fn main() -> void {
    let Some(value) = maybe_value()
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    let Some(value) = maybe_value() else {
        return
    }
}
```

## Fix

For let-else pattern bindings, add `else { ... }`. For constants, start the
declaration with `const`.

No automatic code action is currently provided.

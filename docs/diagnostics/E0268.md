# E0268: Missing Let-Else Body or Const Type Separator

## Explanation

This parser code is used in two nearby contexts:

- after `let Pattern(binding) = value else`, the parser expects `{` to start the
  else body;
- after a constant name, the parser expects `:` before the constant type.

## Error Example

```nomo
package app.main

fn main() -> void {
    let Some(value) = maybe_value() else
        return
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    let Some(value) = maybe_value() else {
        return
    }
}
```

## Fix

Wrap let-else fallback statements in `{ ... }`. For constants, write
`const NAME: Type = value`.

No automatic code action is currently provided.

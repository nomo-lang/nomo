# E0269: Missing Const Initializer or If-Let Statement

## Explanation

This parser code is used when:

- a constant declaration is missing `=` before its value;
- the parser entered the if-let statement path but did not find `if`.

## Error Example

```nomo
package app.main

const LIMIT: i64 10
```

## Correct Code

```nomo
package app.main

const LIMIT: i64 = 10
```

## Fix

For constants, add `=` before the initializer. For if-let statements, start the
statement with `if let`.

## LSP Code Action

No automatic code action is currently provided.

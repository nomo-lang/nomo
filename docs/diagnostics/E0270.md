# E0270: Missing If-Let Let Keyword

## Explanation

An if-let statement starts with `if let`. The parser found `if` but did not find
`let` after it.

## Error Example

```nomo
package app.main

fn main() -> void {
    if Some(value) = maybe_value() {
        return
    }
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    if let Some(value) = maybe_value() {
        return
    }
}
```

## Fix

Insert `let` after `if` when matching a value against a pattern.

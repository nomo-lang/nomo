# E0271: Missing If-Let Initializer Separator

## Explanation

After the if-let pattern, the parser expects `=` before the value being matched.

## Error Example

```nomo
package app.main

fn main() -> void {
    if let Some(value) maybe_value() {
        return
    }
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    if let Some(value) = maybe_value() {
        return
    }
}
```

## Fix

Add `=` between the pattern and the expression.

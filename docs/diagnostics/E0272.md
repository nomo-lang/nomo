# E0272: Missing If-Let Body

## Explanation

After `if let Pattern = value`, the parser expects `{` to start the then body.

## Error Example

```nomo
package app.main

fn main() -> void {
    if let Some(value) = maybe_value()
        return
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    if let Some(value) = maybe_value() {
        return
    }
}
```

## Fix

Wrap the if-let body in `{ ... }`.

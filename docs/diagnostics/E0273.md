# E0273: Missing If-Let Else Body

## Explanation

When an if-let statement has an `else`, the parser expects `{` to start the else
body.

## Error Example

```nomo
package app.main

fn main() -> void {
    if let Some(value) = maybe_value() {
        return
    } else
        return
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    if let Some(value) = maybe_value() {
        return
    } else {
        return
    }
}
```

## Fix

Wrap the if-let else body in `{ ... }`.

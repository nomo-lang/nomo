# E0274: Unsupported Wildcard Import

## Explanation

Wildcard imports are not supported in v0.1. Import a module or concrete public
item instead.

## Error Example

```nomo
package app.main

import std.io.*
```

## Correct Code

```nomo
package app.main

import std.io
```

Or import a concrete item:

```nomo
package app.main

import std.io.println
```

## Fix

Replace `.*` with the module path or a concrete item path.

## LSP Code Action

No automatic code action is currently provided.

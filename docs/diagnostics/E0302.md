# E0302: Duplicate Local Binding

## Explanation

A function scope already contains a binding with the same name. Parameters,
constants imported into the local scope, `let` bindings, loop bindings, match
payload bindings, and generated block bindings must not collide in the same
scope.

## Error Example

```nomo
package app.main

fn main() -> void {
    let count: i64 = 1
    let count: i64 = 2
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    let count: i64 = 1
    let next_count: i64 = 2
}
```

## Fix

Rename one of the bindings or move it into a separate scope where shadowing is
intentional and supported.

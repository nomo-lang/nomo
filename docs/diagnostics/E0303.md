# E0303: Unknown Variable

## Explanation

The compiler could not find a variable with the requested name in the current
scope. Bindings introduced inside blocks, loop bodies, match arms, and if-let
bodies do not escape those bodies.

## Error Example

```nomo
package app.main

fn main() -> void {
    value = 1
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    let mut value: i64 = 0
    value = 1
}
```

## Fix

Declare the variable before using it, check the spelling, or move the use into
the scope where the variable is defined.

# E0305: Unknown Function or Non-Callable Local

## Explanation

The compiler could not resolve a call target as a function, method, builtin, or
enum constructor. This code is also used when a local variable shadows a name and
that local variable is then called like a function.

## Error Example

```nomo
package app.main

fn main() -> void {
    missing()
}
```

## Correct Code

```nomo
package app.main

fn missing() -> void {
}

fn main() -> void {
    missing()
}
```

## Fix

Define the function, import the standard library function you intended to call,
or rename any local binding that shadows the callable name.

## LSP Code Action

No automatic code action is currently provided.

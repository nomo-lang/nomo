# E0306: Duplicate Struct Definition

## Explanation

Two struct declarations in the same compile unit use the same type name.

## Error Example

```nomo
package app.main

struct Point {
    x: i64
}

struct Point {
    y: i64
}
```

## Correct Code

```nomo
package app.main

struct Point {
    x: i64
}

struct Size {
    y: i64
}
```

## Fix

Rename one struct or merge the fields into a single declaration.

## LSP Code Action

No automatic code action is currently provided.

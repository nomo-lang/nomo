# E0307: Duplicate Struct Field

## Explanation

A struct declaration defines the same field name more than once.

## Error Example

```nomo
package app.main

struct Point {
    x: i64
    x: i64
}
```

## Correct Code

```nomo
package app.main

struct Point {
    x: i64
    y: i64
}
```

## Fix

Rename the duplicate field or remove one of the declarations.

No automatic code action is currently provided.

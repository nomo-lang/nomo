# E0308: Unknown Struct Field

## Explanation

The compiler resolved the base value as a struct but could not find the
requested field on that struct type.

## Error Example

```nomo
package app.main

struct Point {
    x: i64
}

fn main() -> void {
    let point: Point = Point { x: 1 }
    let y: i64 = point.y
}
```

## Correct Code

```nomo
package app.main

struct Point {
    x: i64
}

fn main() -> void {
    let point: Point = Point { x: 1 }
    let x: i64 = point.x
}
```

## Fix

Use an existing field name, add the field to the struct declaration, or correct
the type of the base value.

## LSP Code Action

No automatic code action is currently provided.

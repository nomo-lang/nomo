# E0309: Unknown Struct or Impl Target

## Explanation

The compiler could not resolve a struct name used in a struct literal or impl
target. Impl blocks in v0.1 must target a known local struct.

## Error Example

```nomo
package app.main

fn main() -> void {
    let point: Point = Point { x: 1 }
}
```

## Correct Code

```nomo
package app.main

struct Point {
    x: i64
}

fn main() -> void {
    let point: Point = Point { x: 1 }
}
```

## Fix

Define the struct before using it, check the type name spelling, or update the
impl block to target a known local struct.

## LSP Code Action

No automatic code action is currently provided.

# E0310: Missing Struct Literal Field

## Explanation

A struct literal must provide every field declared by the struct. The compiler
reports this when a required field is absent.

## Error Example

```nomo
package app.main

struct Point {
    x: i64
    y: i64
}

fn main() -> void {
    let point: Point = Point { x: 1 }
}
```

## Correct Code

```nomo
package app.main

struct Point {
    x: i64
    y: i64
}

fn main() -> void {
    let point: Point = Point { x: 1, y: 2 }
}
```

## Fix

Add the missing field to the struct literal or remove the field from the struct
declaration if it is no longer part of the type.

## LSP Code Action

No automatic code action is currently provided.

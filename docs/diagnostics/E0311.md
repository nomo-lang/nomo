# E0311: Duplicate Struct Literal Field

## Explanation

A struct literal specified the same field more than once. Each field may appear
only once in a literal.

## Error Example

```nomo
package app.main

struct Point {
    x: i64
}

fn main() -> void {
    let point: Point = Point { x: 1, x: 2 }
}
```

## Correct Code

```nomo
package app.main

struct Point {
    x: i64
}

fn main() -> void {
    let point: Point = Point { x: 1 }
}
```

## Fix

Remove the duplicate field or merge the value into a single field initializer.

No automatic code action is currently provided.

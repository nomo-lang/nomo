# E0313: Duplicate Enum Definition

## Explanation

Two enum declarations in the same compile unit use the same type name.

## Error Example

```nomo
package app.main

enum Color {
    Red
}

enum Color {
    Blue
}
```

## Correct Code

```nomo
package app.main

enum Color {
    Red
}

enum Shade {
    Blue
}
```

## Fix

Rename one enum or merge the variants into a single enum declaration.

## LSP Code Action

No automatic code action is currently provided.

# E0314: Duplicate Enum Variant

## Explanation

An enum declaration defines the same variant name more than once.

## Error Example

```nomo
package app.main

enum Color {
    Red
    Red
}
```

## Correct Code

```nomo
package app.main

enum Color {
    Red
    Blue
}
```

## Fix

Rename or remove the duplicate variant.

## LSP Code Action

No automatic code action is currently provided.

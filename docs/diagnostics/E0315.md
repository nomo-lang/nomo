# E0315: Unknown Enum Variant

## Explanation

The compiler found an enum name but could not find the requested variant on
that enum.

## Error Example

```nomo
package app.main

enum Color {
    Red
}

fn main() -> void {
    let color: Color = Color.Blue
}
```

## Correct Code

```nomo
package app.main

enum Color {
    Red
}

fn main() -> void {
    let color: Color = Color.Red
}
```

## Fix

Use a variant declared on the enum, add the variant, or correct the enum name.

## LSP Code Action

No automatic code action is currently provided.

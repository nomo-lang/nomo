# E0316: Invalid Match Arm Variant Path

## Explanation

Match arms over enum values must identify variants using `Enum.Variant`, or one
of the compiler-supported prelude variants such as `Some`, `None`, `Ok`, and
`Err` when not shadowed.

## Error Example

```nomo
package app.main

enum Color {
    Red
}

fn label(color: Color) -> string {
    return match color {
        Red => "red"
    }
}
```

## Correct Code

```nomo
package app.main

enum Color {
    Red
}

fn label(color: Color) -> string {
    return match color {
        Color.Red => "red"
    }
}
```

## Fix

Qualify the variant with its enum name, or avoid shadowing prelude variant names
when using `Some`, `None`, `Ok`, or `Err`.

## LSP Code Action

No automatic code action is currently provided.

# E0318: Non-Exhaustive Match

## Explanation

Nomo requires matches over enums to cover every variant. The compiler reports
this when at least one variant is missing.

## Error Example

```nomo
package app.main

enum Color {
    Red
    Blue
}

fn label(color: Color) -> string {
    return match color {
        Color.Red => "red"
    }
}
```

## Correct Code

```nomo
package app.main

enum Color {
    Red
    Blue
}

fn label(color: Color) -> string {
    return match color {
        Color.Red => "red"
        Color.Blue => "blue"
    }
}
```

## Fix

Add an arm for every enum variant.

## LSP Code Action

No automatic code action is currently provided.

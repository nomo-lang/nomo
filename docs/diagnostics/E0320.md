# E0320: Enum Variant Requires Payload

## Explanation

The selected enum variant carries a payload, but the expression used the variant
without providing one.

## Error Example

```nomo
package app.main

enum MaybeInt {
    Some(i64)
    None
}

fn main() -> void {
    let value: MaybeInt = MaybeInt.Some
}
```

## Correct Code

```nomo
package app.main

enum MaybeInt {
    Some(i64)
    None
}

fn main() -> void {
    let value: MaybeInt = MaybeInt.Some(1)
}
```

## Fix

Provide the required payload, or use a variant that does not carry a payload.

No automatic code action is currently provided.

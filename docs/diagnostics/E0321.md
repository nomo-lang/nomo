# E0321: Match Arm Must Bind Payload

## Explanation

A match arm targets an enum variant that carries a payload, but the arm pattern
does not bind that payload.

## Error Example

```nomo
package app.main

enum MaybeInt {
    Some(i64)
    None
}

fn value_or_zero(value: MaybeInt) -> i64 {
    return match value {
        MaybeInt.Some => 1
        MaybeInt.None => 0
    }
}
```

## Correct Code

```nomo
package app.main

enum MaybeInt {
    Some(i64)
    None
}

fn value_or_zero(value: MaybeInt) -> i64 {
    return match value {
        MaybeInt.Some(number) => number
        MaybeInt.None => 0
    }
}
```

## Fix

Add a binding name in parentheses after the payload variant.

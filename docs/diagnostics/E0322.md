# E0322: Match Arm Cannot Bind Unit Variant

## Explanation

A match arm provides a binding for an enum variant that does not carry a payload.

## Error Example

```nomo
package app.main

enum Color {
    Red
}

fn label(color: Color) -> string {
    return match color {
        Color.Red(value) => "red"
    }
}
```

## Correct Code

```nomo
package app.main

enum Color {
    Red
}

fn label(color: Color) -> string {
    return match color {
        Color.Red => "red"
    }
}
```

## Fix

Remove the binding from variants that do not carry payloads.

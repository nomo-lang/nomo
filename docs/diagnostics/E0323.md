# E0323: Enum Variant Does Not Accept Payload

## Explanation

The selected enum variant is a unit variant, but the code called it with a
payload.

## Error Example

```nomo
package app.main

enum Color {
    Red
}

fn main() -> void {
    let color: Color = Color.Red(1)
}
```

## Correct Code

```nomo
package app.main

enum Color {
    Red
}

fn main() -> void {
    let color: Color = Color.Red
}
```

## Fix

Remove the payload argument, or declare the variant with a payload type.

No automatic code action is currently provided.

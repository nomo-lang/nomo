# E0402: Value Cannot Be Formatted

## Explanation

`std.fmt` and the `std.io` print helpers can format primitive scalar values.
Local non-generic structs must implement `fmt.Display` for `{}` and ordinary
printing, or `fmt.Debug` for `{:?}`.

## Error Example

```nomo
package app.main

import std.io

struct User {
    name: string
}

fn main() -> void {
    let user: User = User { name: "Nomo" }
    io.println(user)
}
```

## Correct Code

```nomo
package app.main

import std.fmt
import std.io

struct User {
    name: string
}

impl fmt.Display for User {
    fn to_string(self) -> string {
        return self.name
    }
}

fn main() -> void {
    let user: User = User { name: "Nomo" }
    io.println(user)
}
```

## Fix

Implement the interface named in the diagnostic, or convert the value to a
supported scalar string before formatting it.

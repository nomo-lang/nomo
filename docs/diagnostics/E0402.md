# E0402: Invalid IO Print Arguments

## Explanation

The v0.1 `io.print`, `io.println`, `io.eprint`, and `io.eprintln` helpers expect
exactly one `string` argument.

## Error Example

```nomo
package app.main

import std.io

fn main() -> void {
    io.println(1)
}
```

## Correct Code

```nomo
package app.main

import std.io

fn main() -> void {
    io.println("1")
}
```

## Fix

Pass one string argument. Convert values to string explicitly before printing.

No automatic code action is currently provided.

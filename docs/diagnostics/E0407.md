# E0407: Invalid Call or Constructor Arguments

## Explanation

A function, standard-library helper, method, or enum constructor was called with
the wrong number of arguments or without required generic type arguments.

## Error Example

```nomo
package app.main

fn identity<T>(value: T) -> T {
    return value
}

fn main() -> void {
    let value: i32 = identity(1)
}
```

## Correct Code

```nomo
package app.main

fn identity<T>(value: T) -> T {
    return value
}

fn main() -> void {
    let value: i32 = identity<i32>(1)
}
```

## Fix

Match the callable's expected arity and provide explicit type arguments for
generic functions or constructors when required.

## LSP Code Action

No automatic code action is currently provided.

# E0408: Invalid Format Template

## Explanation

`fmt.format` requires a compile-time string literal containing only `{}`,
`{:?}`, `{{`, and `}}` formatting forms. The number of supplied values must
match the number of placeholders.

## Error Example

```nomo
package app.main

import std.fmt

fn main() -> void {
    let name = "Nomo"
    let message = fmt.format("Hello, {name}", name)
}
```

## Correct Code

```nomo
package app.main

import std.fmt

fn main() -> void {
    let name = "Nomo"
    let message = fmt.format("Hello, {}", name)
}
```

## Fix

Replace named or C-style placeholders with `{}` or `{:?}`, escape literal
braces as `{{` and `}}`, and pass exactly one value per placeholder.

No automatic code action is currently provided.

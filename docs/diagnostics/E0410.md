# E0410: Recursive Value Type

## Explanation

A nominal type is recursively embedded by value. Direct value recursion would
make the type layout infinite, so v0.1 rejects it. `Array<T>` is treated as an
indirection boundary and may be used for recursive shapes.

## Error Example

```nomo
package app.main

struct Node {
    next: Node
}
```

## Correct Code

```nomo
package app.main

import std.array

struct Node {
    children: Array<Node>
}
```

## Fix

Break the value recursion by using an indirection boundary supported by the
language, such as `Array<T>`, or redesign the type so it does not contain itself
by value.

## LSP Code Action

No automatic code action is currently provided.

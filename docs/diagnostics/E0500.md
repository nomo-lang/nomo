# E0500: Missing Mut Call Argument Marker

## Explanation

A function or method parameter was declared as `mut`, but the call passed the
argument without the `mut` marker. Nomo requires mutable borrows to be explicit
at both the callee and caller.

## Error Example

```nomo
package app.main

fn inspect(mut value: i64) -> i64 {
    return value
}

fn main() -> void {
    let mut count: i64 = 41
    let answer: i64 = inspect(count)
}
```

## Correct Code

```nomo
package app.main

fn inspect(mut value: i64) -> i64 {
    return value
}

fn main() -> void {
    let mut count: i64 = 41
    let answer: i64 = inspect(mut count)
}
```

## Fix

Pass the argument with `mut` when mutation is intentional, or change the
parameter to an immutable parameter if the callee does not need to mutate it.

## LSP Code Action

No automatic code action is currently provided.

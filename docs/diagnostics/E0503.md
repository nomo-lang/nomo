# E0503: Invalid Mut Call Argument Target

## Explanation

The `mut` call argument marker can only borrow a local variable, parameter, or
field path. It cannot be used on enum payload pseudo-bindings or other values
that are not mutable storage locations.

## Error Example

```nomo
package app.main

fn inspect(mut value: string) -> void {
}

fn main() -> void {
    let value: Option<string> = Some("nomo")
    match value {
        Some(text) => {
            inspect(mut text)
        }
        None => {
        }
    }
}
```

## Correct Code

```nomo
package app.main

fn inspect(mut value: string) -> void {
}

fn main() -> void {
    let value: Option<string> = Some("nomo")
    match value {
        Some(text) => {
            let mut copy: string = text
            inspect(mut copy)
        }
        None => {
        }
    }
}
```

## Fix

Store the value in a mutable local binding first, then pass that binding or one
of its fields as `mut`.

No automatic code action is currently provided.

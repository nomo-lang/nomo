# E0504: Unexpected Mut Call Argument Marker

## Explanation

The call used `mut`, but the corresponding parameter is immutable. Caller and
callee mutability must match exactly.

## Error Example

```nomo
package app.main

fn inspect(value: i64) -> i64 {
    return value
}

fn main() -> void {
    let mut count: i64 = 41
    let answer: i64 = inspect(mut count)
}
```

## Correct Code

```nomo
package app.main

fn inspect(value: i64) -> i64 {
    return value
}

fn main() -> void {
    let count: i64 = 41
    let answer: i64 = inspect(count)
}
```

## Fix

Remove `mut` from the call argument, or change the callee parameter to `mut` if
the function really needs a mutable borrow.

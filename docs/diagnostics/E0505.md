# E0505: Mut Marker Outside Call Argument

## Explanation

The `mut` marker is only valid when a call argument is matched with a `mut`
parameter. It is not a normal value expression, so builtins and other expression
positions reject it.

## Error Example

```nomo
package app.main

import std.io

fn main() -> void {
    let mut text: string = "hello"
    io.println(mut text)
}
```

## Correct Code

```nomo
package app.main

fn inspect(mut value: i64) -> i64 {
    return value
}

fn main() -> void {
    let mut count: i64 = 41
    let value: i64 = inspect(mut count)
}
```

## Fix

Use `mut` only when passing an argument to a user-defined `mut` parameter. For
ordinary expressions and builtin calls, refer to the variable without `mut`.

# E0510: Break Outside Loop

## Explanation

`break` can only appear inside a `for` loop body. It cannot be used at the top
level of a function, in a non-loop block, or in a script body outside a loop.

## Error Example

```nomo
package app.main

fn main() -> void {
    break
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    for {
        break
    }
}
```

## Fix

Move `break` inside the loop it should exit, or replace it with `return` or
`panic` when the intended behavior is to leave the current function.

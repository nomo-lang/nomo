# E0511: Continue Outside Loop

## Explanation

`continue` can only appear inside a `for` loop body. It cannot be used at the top
level of a function, in a non-loop block, or in a script body outside a loop.

## Error Example

```nomo
package app.main

fn main() -> void {
    continue
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    for {
        continue
    }
}
```

## Fix

Move `continue` inside the loop it should advance, or remove it if there is no
enclosing loop.

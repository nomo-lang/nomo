# E0607: Cyclic Module Import

## Explanation

The module graph contains an import cycle. A module cannot directly or
indirectly import itself because public API merging requires an acyclic module
order.

The diagnostic includes the complete cycle path, such as
`app.a -> app.b -> app.a`.

## Error Example

```nomo
package app.a

import app.b
```

when `app.b` imports `app.a`.

## Correct Code

```nomo
package app.a

import app.shared
```

Move shared declarations into a third module that both original modules can
import without forming a cycle.

## Fix

Remove one edge in the reported import path, or extract the shared API into a
lower-level module.

No automatic code action is currently provided.

# E0903: Module Not Found

## Explanation

The compiler resolved an `import` root, but no source file matched the imported
module path. Local modules are searched under the project `src/` directory, and
path or git dependencies are searched under the dependency `src/` directory.

For example, `import app.math` looks for `src/math.nomo` first and then
`src/math/main.nomo`.

## Error Example

```nomo
package app.main

import app.missing

fn main() -> void {
}
```

## Correct Code

```nomo
package app.main

import app.math

fn main() -> void {
}
```

with one of these files:

```text
src/math.nomo
src/math/main.nomo
```

## Fix

Create the missing module file, rename the import to an existing module, or
move the module file so it follows the current module mapping rules.

## LSP Code Action

No automatic code action is currently provided.

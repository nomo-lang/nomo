# E1100: Invalid or Unsupported Attribute

## Explanation

Attributes use the `#[name]` syntax. v0.1 currently supports `#[test]` on
top-level functions. Other attribute names or malformed attribute syntax are
rejected.

## Error Example

```nomo
package app.main

#[bench]
fn fast() -> void {
}
```

## Correct Code

```nomo
package app.main

#[test]
fn fast() -> void {
}
```

## Fix

Use `#[test]` only on top-level test functions, and keep the syntax exactly
`#[test]` followed by a newline and function declaration.

## LSP Code Action

No automatic code action is currently provided.

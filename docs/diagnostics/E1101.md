# E1101: Invalid Test Function Signature

## Explanation

`#[test]` functions are discovered and run by `nomo test`. They must be simple
top-level functions: no `main` name, no generic type parameters, no parameters,
and a `void` return type.

## Error Example

```nomo
package app.main

#[test]
fn bad(value: i32) -> void {
}
```

## Correct Code

```nomo
package app.main

#[test]
fn good() -> void {
}
```

## Fix

Remove parameters and type parameters from the test function, avoid naming it
`main`, and make it return `void`.

# E1500: Expected Interface Declaration

## Explanation

The parser was reading an interface declaration but did not find the `interface`
keyword where the declaration must start.

## Error Example

```nomo
package app.main

pub Display {
    fn to_string(self) -> string
}
```

## Correct Code

```nomo
package app.main

pub interface Display {
    fn to_string(self) -> string
}
```

## Fix

Add the `interface` keyword before the interface name.

# E1501: Missing Interface Body

## Explanation

An interface declaration must open a method-signature block with `{`.

## Error Example

```nomo
package app.main

interface Display
    fn to_string(self) -> string
}
```

## Correct Code

```nomo
package app.main

interface Display {
    fn to_string(self) -> string
}
```

## Fix

Place `{` after the interface name.

No automatic code action is currently provided.

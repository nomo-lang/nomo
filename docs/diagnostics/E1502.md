# E1502: Unterminated Interface Body

## Explanation

The parser reached the end of the file before seeing the closing `}` for an
interface declaration.

## Error Example

```nomo
package app.main

interface Display {
    fn to_string(self) -> string
```

## Correct Code

```nomo
package app.main

interface Display {
    fn to_string(self) -> string
}
```

## Fix

Add the missing `}` after the interface method signatures.

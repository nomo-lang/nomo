# E1503: Invalid Interface Member

## Explanation

Only method signatures are allowed inside a v0.1 interface body.

## Error Example

```nomo
package app.main

interface Display {
    const NAME: string = "display"
}
```

## Correct Code

```nomo
package app.main

interface Display {
    fn to_string(self) -> string
}
```

## Fix

Replace the member with a `fn` method signature or move the declaration outside
the interface.

No automatic code action is currently provided.

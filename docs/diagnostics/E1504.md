# E1504: Expected Interface Method

## Explanation

An interface member must start with `fn`.

## Error Example

```nomo
package app.main

interface Display {
    to_string(self) -> string
}
```

## Correct Code

```nomo
package app.main

interface Display {
    fn to_string(self) -> string
}
```

## Fix

Add `fn` before the interface method name.

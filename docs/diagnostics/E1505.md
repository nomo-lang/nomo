# E1505: Missing Interface Method Parameters

## Explanation

An interface method signature must include a parameter list after the method
name, even when the list is empty.

## Error Example

```nomo
package app.main

interface Display {
    fn to_string -> string
}
```

## Correct Code

```nomo
package app.main

interface Display {
    fn to_string(self) -> string
}
```

## Fix

Add `(...)` after the interface method name.

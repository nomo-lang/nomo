# E1506: Unsatisfied Generic Interface Bound

## Explanation

A constrained generic function names an unknown or invalid interface, uses an
operation that is not available through the declared interface, or receives a
concrete type argument that does not implement the required interface. Nomo
checks the generic body against the interface contract, then resolves valid
calls with static dispatch during monomorphization.

## Error Example

```nomo
package app.main

interface Display {
    fn to_string(self) -> string
}

struct User {
    name: string
}

fn render<T: Display>(value: T) -> string {
    return "missing impl"
}

fn main() -> void {
    let user: User = User { name: "Nomo" }
    let text: string = render<User>(user)
}
```

## Correct Code

```nomo
package app.main

interface Display {
    fn to_string(self) -> string
}

struct User {
    name: string
}

impl Display for User {
    fn to_string(self) -> string {
        return self.name
    }
}

fn render<T: Display>(value: T) -> string {
    return value.to_string()
}
```

## Fix

Declare the interface before the generic function and add a matching
`impl Interface for Type` for every concrete type argument used at a call site.

## LSP Code Action

No automatic implementation generation is currently provided.

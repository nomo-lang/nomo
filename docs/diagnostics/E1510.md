# E1510: Expected Extern Declaration

## Explanation

The parser was reading an extern declaration but did not find the `extern`
keyword where the declaration must start.

## Error Example

```nomo
package app.main

"C" {
    fn puts(message: string) -> i32
}
```

## Correct Code

```nomo
package app.main

extern "C" {
    fn puts(message: string) -> i32
}
```

## Fix

Add `extern` before the ABI string.

## LSP Code Action

No automatic code action is currently provided.

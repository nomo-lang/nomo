# E1511: Unsupported Extern ABI

## Explanation

v0.1 only accepts `extern "C"` blocks. Other ABI strings are rejected.

## Error Example

```nomo
package app.main

extern "system" {
    fn puts(message: string) -> i32
}
```

## Correct Code

```nomo
package app.main

extern "C" {
    fn puts(message: string) -> i32
}
```

## Fix

Use the exact ABI string `"C"`.

## LSP Code Action

No automatic code action is currently provided.

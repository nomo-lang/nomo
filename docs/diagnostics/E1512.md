# E1512: Missing Extern Block Body

## Explanation

An extern declaration must open a block of external function declarations with
`{`.

## Error Example

```nomo
package app.main

extern "C"
    fn puts(message: string) -> i32
}
```

## Correct Code

```nomo
package app.main

extern "C" {
    fn puts(message: string) -> i32
}
```

## Fix

Place `{` after the ABI string.

## LSP Code Action

No automatic code action is currently provided.

# E1513: Unterminated Extern Block

## Explanation

The parser reached the end of the file before seeing the closing `}` for an
extern block.

## Error Example

```nomo
package app.main

extern "C" {
    fn puts(message: string) -> i32
```

## Correct Code

```nomo
package app.main

extern "C" {
    fn puts(message: string) -> i32
}
```

## Fix

Add the missing `}` after the extern function declarations.

No automatic code action is currently provided.

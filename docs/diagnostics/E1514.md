# E1514: Invalid Extern Declaration

## Explanation

Only function declarations are allowed inside a v0.1 extern block.

## Error Example

```nomo
package app.main

extern "C" {
    const EXIT_SUCCESS: i32 = 0
}
```

## Correct Code

```nomo
package app.main

extern "C" {
    fn puts(message: string) -> i32
}
```

## Fix

Replace the member with an extern `fn` declaration or move it outside the extern
block.

## LSP Code Action

No automatic code action is currently provided.

# E1515: Expected Extern Function

## Explanation

An extern block member must start with `fn`.

## Error Example

```nomo
package app.main

import std.ffi

extern "C" {
    puts(message: string) -> i32
}
```

## Correct Code

```nomo
package app.main

extern "C" {
    fn puts(message: CString) -> i32
}
```

## Fix

Add `fn` before the extern function name.

No automatic code action is currently provided.

# E1516: Missing Extern Function Parameters

## Explanation

An extern function declaration must include a parameter list after the function
name, even when the list is empty.

## Error Example

```nomo
package app.main

import std.ffi

extern "C" {
    fn puts -> i32
}
```

## Correct Code

```nomo
package app.main

extern "C" {
    fn puts(message: CString) -> i32
}
```

## Fix

Add `(...)` after the extern function name.

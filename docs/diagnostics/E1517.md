# E1517: Expected Unsafe Block

## Explanation

The parser was reading an unsafe block but did not find the `unsafe` keyword
where the block must start.

## Error Example

```nomo
package app.main

import std.ffi

extern "C" {
    fn puts(message: CString) -> i32
}

fn main() -> void {
    let message: CString = CString.from_string("hello")
    {
        puts(message)
    }
}
```

## Correct Code

```nomo
package app.main

import std.ffi

extern "C" {
    fn puts(message: CString) -> i32
}

fn main() -> void {
    let message: CString = CString.from_string("hello")
    unsafe {
        puts(message)
    }
}
```

## Fix

Use `unsafe { ... }` around the FFI boundary operation.

No automatic code action is currently provided.

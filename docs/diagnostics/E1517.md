# E1517: Expected Unsafe Block

## Explanation

The parser was reading an unsafe block but did not find the `unsafe` keyword
where the block must start.

## Error Example

```nomo
package app.main

fn main() -> void {
    {
        puts("hello")
    }
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    unsafe {
        puts("hello")
    }
}
```

## Fix

Use `unsafe { ... }` around the FFI boundary operation.

No automatic code action is currently provided.

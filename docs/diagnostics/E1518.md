# E1518: Missing Unsafe Block Body

## Explanation

An unsafe block must open its body with `{`.

## Error Example

```nomo
package app.main

import std.ffi

extern "C" {
    fn puts(message: CString) -> i32
}

fn main() -> void {
    let message: CString = CString.from_string("hello")
    unsafe puts(message)
}
```

## Correct Code

```nomo
package app.main

import std.ffi

extern "C" {
    fn puts(message: CString) -> i32
}

fn main() -> void {
    let message: CString = CString.from_string("hello")
    unsafe {
        puts(message)
    }
}
```

## Fix

Place `{` after `unsafe` and close the block with `}`.

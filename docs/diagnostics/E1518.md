# E1518: Missing Unsafe Block Body

## Explanation

An unsafe block must open its body with `{`.

## Error Example

```nomo
package app.main

fn main() -> void {
    unsafe puts("hello")
}
```

## Correct Code

```nomo
package app.main

fn main() -> void {
    unsafe {
        puts("hello")
    }
}
```

## Fix

Place `{` after `unsafe` and close the block with `}`.

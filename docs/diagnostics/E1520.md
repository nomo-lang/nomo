# E1520: Invalid Opaque Handle Declaration

## Explanation

An `extern opaque type` declaration is not in the supported form.

## Error Example

```nomo
extern handle type Window
```

## Correct Code

```nomo
extern opaque type Window
```

## Fix

Use `extern opaque type Name` and place the declaration at top level. An
optional `release FunctionName` clause must name the matching C release
function.

## LSP Code Action

No automatic code action is currently provided.

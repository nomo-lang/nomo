# E1522: Opaque Handle Construction

## Explanation

Opaque handles represent values owned by C. Nomo code cannot construct them
with a struct literal.

## Error Example

```nomo
extern opaque type Window
let window: Window = Window {}
```

## Correct Code

```nomo
extern "C" {
    fn open_window() -> Window
}
let window: Window = unsafe { open_window() }
```

## Fix

Obtain handles from an extern function and pass them back through supported
FFI operations. Do not inspect or construct their representation in Nomo.

## LSP Code Action

No automatic code action is currently provided.

# E1523: Invalid Opaque Handle Release Function

## Explanation

The release function attached to an opaque handle is missing or has an
incompatible extern C signature.

## Error Example

```nomo
extern opaque type Window release close_window
extern "C" {
    fn close_window(handle: Window) -> i32
}
```

## Correct Code

```nomo
extern opaque type Window release close_window
extern "C" {
    fn close_window(handle: Owned<Window>) -> void
}
```

## Fix

Declare the release function in an `extern "C"` block with one immutable
`Owned<Handle>` parameter and a `void` return type.

## LSP Code Action

No automatic code action is currently provided.

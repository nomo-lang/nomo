# E1524: Invalid Extern C Callback Type

## Explanation

An extern callback type is missing the supported `extern "C" fn(...) -> ...`
shape or has malformed parameter separators or return syntax.

## Error Example

```nomo
type Callback = extern "Rust" fn(i32) -> void
```

## Correct Code

```nomo
type Callback = extern "C" fn(i32) -> void
```

## Fix

Use the exact C ABI spelling, separate callback parameters with commas, and
provide one return type after `->`.

## LSP Code Action

No automatic code action is currently provided.

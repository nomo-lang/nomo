# E1530: Invalid repr(C) Layout

## Explanation

The fields of a `#[repr(C)]` struct are not representable with the supported
deterministic C ABI layout.

## Error Example

```nomo
#[repr(C)]
struct Message {
    text: string
}
```

## Correct Code

```nomo
#[repr(C)]
struct Message {
    code: i32
    handle: Opaque
}
```

## Fix

Use fixed-width scalars, handles, nullable handles, or another non-generic
`#[repr(C)]` struct. C-layout structs cannot be generic or empty.

## LSP Code Action

No automatic code action is currently provided.
